# AI--Driven-Threat-Detection
Cyber-security Researcher | Security Monitoring

## Overview
An AI-powered security monitoring tool designed 
to automate threat detection for Security 
Operations Centres (SOC).

## Problem Statement
SOC teams face overwhelming volumes of security 
alerts daily. This project uses AI and machine 
learning to automate threat detection and reduce 
response times.

## Features (In Development)
- Real-time log monitoring
- AI-powered anomaly detection
- Automated threat alerts
- Security dashboard

## Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit
